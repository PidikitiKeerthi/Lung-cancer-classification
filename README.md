Lung Cancer Classification

Overview : This project focuses on classifying lung cancer images (CT / histopathology) using machine learning and deep learning models.

Objective:
Classify lung cancer into different classes
Compare ML and DL models to find the best performer

Models Used :
Deep Learning: CNN, VGG16, Swin Transformer
Machine Learning: Random Forest ⭐, Decision Tree, KNN, SVM

Best Model : Random Forest (~93% accuracy)

Tech Stack :
Python
TensorFlow / PyTorch
Scikit-learn, OpenCV
Matplotlib, Seaborn

Method :
Image preprocessing (resize, normalize)
Feature extraction (VGG16 / Swin)
Model training & evaluation

Results :
Random Forest: 93%
Decision Tree: 90%
Swin Transformer: 83%
VGG16: 80%

Evaluation :
Accuracy, Precision, Recall, F1-score
Confusion Matrix, ROC Curve

Future Work :
Use larger datasets
Apply Explainable AI
Deploy as web app
